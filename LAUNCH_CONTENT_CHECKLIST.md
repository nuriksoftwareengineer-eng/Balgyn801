# LAUNCH_CONTENT_CHECKLIST

**Чек-лист наполнения магазина перед открытием**  
**Бренд:** BALGYN · Алматы · 2026

Все задачи выполняются в админ-панели `/admin` или напрямую в коде/БД, если указано.

---

## 1. Категории

- [ ] Войти в `/admin/catalog` → вкладка «Группы»
- [ ] Создать активные группы: **Аниме, Игры, Музыка, Спорт, Кино**
- [ ] Задать корректный `slug` для каждой группы (латиница, без пробелов)
- [ ] Задать `sort_order` (порядок отображения)
- [ ] Открыть `/catalog` в браузере — убедиться, что группы отображаются

---

## 2. Коллекции

- [ ] Для каждой активной группы создать минимум 1 коллекцию
- [ ] Задать `slug`, название, `sort_order`
- [ ] Загрузить **cover-изображение** коллекции (1:1 или 4:3, min 800px)
- [ ] Загрузить **banner-изображение** коллекции (16:9, min 1200px) — опционально
- [ ] Написать краткое описание коллекции (1–2 предложения)
- [ ] Открыть `/catalog/{group}` — убедиться, что коллекции видны

---

## 3. Дизайны

- [ ] Для каждой активной коллекции создать минимум 1 дизайн в статусе DRAFT
- [ ] Заполнить: название, slug, описание (3–5 предложений о дизайне и технике вышивки)
- [ ] Перейти к вариантам дизайна (кнопка «Варианты»)
- [ ] Опубликовать дизайн (кнопка «Опубликовать»), только после заполнения фото и вариантов

---

## 4. Варианты (garments) и цены

- [ ] Для каждого дизайна добавить нужные варианты: Hoodie / T-Shirt / Sweatshirt
- [ ] Для каждого варианта установить цену в KZT
- [ ] Для каждого варианта отметить доступные **цвета**
- [ ] Для каждого варианта отметить доступные **размеры**
- [ ] Открыть страницу дизайна — убедиться, что варианты, цвета, размеры корректно выбираются

---

## 5. Фотографии

- [ ] Подготовить минимум **3 фото** на дизайн:
  - [ ] Фото 1: главное (на белом/чёрном фоне, вышивка крупно)
  - [ ] Фото 2: детальный план (макро нитей)
  - [ ] Фото 3: на человеке (front)
- [ ] Загрузить фото через `/admin/catalog/{id}/variants` → раздел изображений
- [ ] Убедиться, что **главное фото** установлено (поле `main_image_url`)
- [ ] Проверить: `/catalog/{group}/{collection}/{design}` — галерея отображается

---

## 6. Размеры

- [ ] Убедиться, что в БД есть записи: **S · M · L · XL · XXL**
- [ ] При необходимости добавить через `/admin` или сид: `INSERT INTO sizes (label) VALUES ('S'),('M'),...`
- [ ] Для каждого варианта дизайна отметить доступные размеры

---

## 7. Цвета

- [ ] Убедиться, что в БД есть: **Black · White · Navy · Red**
- [ ] Добавить при необходимости: Beige / Cream (#F5F0E8)
- [ ] Для каждого варианта дизайна отметить доступные цвета
- [ ] Проверить: свотчи цветов отображаются на странице дизайна

---

## 8. Остатки (inventory)

- [ ] Для каждой комбинации **дизайн + вариант + цвет + размер** установить остаток
- [ ] Минимальный остаток для показа товара: **≥ 1**
- [ ] Стартовый остаток: 3–10 единиц (зависит от производственного запаса)
- [ ] Проверить: кнопка «В корзину» активна, кнопка неактивна при 0 на складе
